## 1. 浏览与加入课程

### 1.1 查看可用课程列表

**接口**：`GET /api/courses`

```bash
# 浏览所有已发布课程（不需要登录）
curl -X GET "http://localhost:8080/api/courses?page=1&size=10"

# 按关键词搜索课程（需要登录）
curl -X GET "http://localhost:8080/api/courses?keyword=Java&page=1&size=10" \
  -H "Authorization: Bearer ${TOKEN}"
```

**响应示例**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "records": [
      {
        "id": 7,
        "title": "Java 程序设计基础",
        "description": "本课程系统讲解 Java 语言核心特性...",
        "teacherName": "李老师",
        "studentCount": 85,
        "status": "ACTIVE",
        "coverUrl": "http://localhost:9000/course-buddy/courses/covers/7.jpg",
        "createdAt": "2024-01-10T08:00:00"
      }
    ],
    "total": 32,
    "size": 10,
    "current": 1,
    "pages": 4
  }
}
```

### 1.2 查看课程详情

**接口**：`GET /api/courses/{courseId}`

```bash
curl -X GET http://localhost:8080/api/courses/7 \
  -H "Authorization: Bearer ${TOKEN}"
```

**响应示例**：
```json
{
  "code": 200,
  "data": {
    "id": 7,
    "title": "Java 程序设计基础",
    "description": "本课程系统讲解 Java 语言核心特性，包括面向对象、集合框架、IO 流等",
    "syllabus": "第1章：Java 入门\n第2章：面向对象...",
    "teacherId": 3,
    "teacherName": "李老师",
    "lessonCount": 16,
    "studentCount": 85,
    "isEnrolled": false,
    "status": "ACTIVE"
  }
}
```

### 1.3 加入课程

**接口**：`POST /api/enrollments`

```bash
curl -X POST http://localhost:8080/api/enrollments \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{"courseId": 7}'
```

**响应**：
```json
{
  "code": 200,
  "message": "成功加入课程",
  "data": {
    "enrollmentId": 156,
    "courseId": 7,
    "courseTitle": "Java 程序设计基础",
    "enrolledAt": "2024-01-15T14:30:00",
    "status": "ACTIVE"
  }
}
```

**注意事项**：
- 需要教师或管理员开放选课权限（课程权限 L1-L4 体系）
- 部分课程可能需要邀请码或审核
- 同一课程不能重复加入

### 1.4 查看已加入的课程

**接口**：`GET /api/enrollments/my`

```bash
curl -X GET http://localhost:8080/api/enrollments/my \
  -H "Authorization: Bearer ${TOKEN}"
```

### 1.5 退出课程

**接口**：`DELETE /api/enrollments/{enrollmentId}`

```bash
curl -X DELETE http://localhost:8080/api/enrollments/156 \
  -H "Authorization: Bearer ${TOKEN}"
```

> 退出课程后，您在该课程中的笔记和提交记录仍会保留在您的账号中，但将无法再访问课程内容。

## 3. 笔记功能

### 3.1 创建笔记

**接口**：`POST /api/notes`

笔记支持 Markdown 格式，可以关联到具体的课程和课节：

```bash
curl -X POST http://localhost:8080/api/notes \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Java 面向对象笔记",
    "content": "## 核心概念\n\n### 封装\n...",
    "courseId": 7,
    "lessonId": 105,
    "categoryId": null,
    "tags": ["Java", "OOP", "期末复习"]
  }'
```

**响应**：
```json
{
  "code": 200,
  "message": "笔记创建成功",
  "data": {
    "id": 88,
    "title": "Java 面向对象笔记",
    "courseId": 7,
    "createdAt": "2024-01-15T15:00:00",
    "version": 1
  }
}
```

### 3.2 更新笔记（支持乐观锁）

**接口**：`PUT /api/notes/{noteId}`

```bash
curl -X PUT http://localhost:8080/api/notes/88 \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Java 面向对象笔记（更新）",
    "content": "## 核心概念\n\n### 封装\n封装是将数据和操作数据的方法绑定在一起...",
    "version": 1
  }'
```

> `version` 字段用于乐观锁控制（V10 迁移添加），防止多设备同时编辑时的数据冲突。每次更新后 version 会自动加 1。

### 3.3 查看笔记列表

```bash
# 查看我的所有笔记
curl -X GET "http://localhost:8080/api/notes?page=1&size=20" \
  -H "Authorization: Bearer ${TOKEN}"

# 按课程筛选笔记
curl -X GET "http://localhost:8080/api/notes?courseId=7&page=1&size=20" \
  -H "Authorization: Bearer ${TOKEN}"

# 按标签筛选
curl -X GET "http://localhost:8080/api/notes?tag=期末复习&page=1&size=20" \
  -H "Authorization: Bearer ${TOKEN}"
```

### 3.4 查看笔记历史版本

**接口**：`GET /api/notes/{noteId}/versions`

```bash
curl -X GET http://localhost:8080/api/notes/88/versions \
  -H "Authorization: Bearer ${TOKEN}"
```

**响应**：
```json
{
  "code": 200,
  "data": [
    {"versionNumber": 2, "updatedAt": "2024-01-15T16:00:00", "changesSummary": "更新封装章节"},
    {"versionNumber": 1, "updatedAt": "2024-01-15T15:00:00", "changesSummary": "初始创建"}
  ]
}
```

### 3.5 导出笔记

**接口**：`GET /api/notes/{noteId}/export?format=pdf`（支持 `pdf` 和 `markdown`）

```bash
# 导出为 PDF
curl -X GET "http://localhost:8080/api/notes/88/export?format=pdf" \
  -H "Authorization: Bearer ${TOKEN}" \
  -o "Java面向对象笔记.pdf"

# 导出为 Markdown
curl -X GET "http://localhost:8080/api/notes/88/export?format=markdown" \
  -H "Authorization: Bearer ${TOKEN}" \
  -o "Java面向对象笔记.md"
```

### 3.6 分享笔记

**接口**：`POST /api/notes/{noteId}/share`

```bash
# 创建分享链接（永不过期）
curl -X POST http://localhost:8080/api/notes/88/share \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{"ttlSeconds": 0}'

# 创建 7 天有效的分享链接
curl -X POST http://localhost:8080/api/notes/88/share \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{"ttlSeconds": 604800}'
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "shareToken": "abc123xyz456",
    "shareUrl": "/api/notes/shared/abc123xyz456",
    "expiresAt": "2024-01-22T15:00:00"
  }
}
```

### 3.7 使用 OCR 识别图片中的文字

**接口**：`POST /api/ocr/recognize`

```bash
curl -X POST http://localhost:8080/api/ocr/recognize \
  -H "Authorization: Bearer ${TOKEN}" \
  -F "image=@/path/to/handwritten_notes.jpg"
```

## 4. 课程讨论与问答

### 4.1 查看课程讨论

**接口**：`GET /api/discussions?courseId={courseId}`

```bash
curl -X GET "http://localhost:8080/api/discussions?courseId=7&page=1&size=20" \
  -H "Authorization: Bearer ${TOKEN}"
```

### 4.2 发布讨论/提问

**接口**：`POST /api/discussions`

```bash
# 发布新讨论/问题
curl -X POST http://localhost:8080/api/discussions \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "courseId": 7,
    "title": "关于 Java 中 == 和 equals() 的区别",
    "content": "老师您好，我在学习第3章时对 == 和 equals() 的区别有些疑惑，请问...",
    "type": "QUESTION"
  }'
```

**响应**：
```json
{
  "code": 200,
  "message": "发布成功",
  "data": {
    "id": 321,
    "courseId": 7,
    "title": "关于 Java 中 == 和 equals() 的区别",
    "authorName": "student_zhangsan",
    "createdAt": "2024-01-15T16:30:00"
  }
}
```

### 4.3 回复讨论

**接口**：`POST /api/discussions`（`parentId` 不为空时为回复）

```bash
curl -X POST http://localhost:8080/api/discussions \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "courseId": 7,
    "parentId": 321,
    "content": "同学你好，== 比较的是引用地址，equals() 比较的是内容...",
    "type": "REPLY"
  }'
```

### 4.4 使用 AI 助手解答问题

**接口**：`POST /api/ai/ask`

```bash
curl -X POST http://localhost:8080/api/ai/ask \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "question": "Java 中 final 关键字有哪些用法？",
    "courseId": 7,
    "context": "当前学习到第5章继承与多态"
  }'
```

> AI 助手基于讯飞星火大模型，相同问题的回答会被缓存 1 小时以提升响应速度。

## 5. 协作功能

### 5.1 创建或加入协作文档

Course Buddy 支持多人实时协作编辑笔记，基于 WebSocket 实现：

**接口**：`POST /api/collaboration/sessions`

```bash
curl -X POST http://localhost:8080/api/collaboration/sessions \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{"noteId": 88}'
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "sessionId": "collab_session_99",
    "wsUrl": "ws://localhost:8080/ws/collaboration/collab_session_99",
    "noteId": 88,
    "activeUsers": ["student_zhangsan"]
  }
}
```

### 5.2 WebSocket 连接协作会话

```javascript
// 前端连接协作 WebSocket
const ws = new WebSocket(
  `ws://localhost:8080/ws/collaboration/collab_session_99?token=${token}`
);

ws.onopen = () => {
  console.log('协作会话已连接');
  // 发送加入消息
  ws.send(JSON.stringify({
    type: 'JOIN',
    sessionId: 'collab_session_99',
    userId: 42,
    username: 'student_zhangsan'
  }));
};

ws.onmessage = (event) => {
  const msg = JSON.parse(event.data);
  // 处理协作消息（文档变更、用户加入/离开等）
  if (msg.type === 'CONTENT_CHANGE') {
    updateDocument(msg.content);
  }
};
```

### 5.3 团队功能

```bash
# 查看我所在的团队
curl -X GET http://localhost:8080/api/teams/my \
  -H "Authorization: Bearer ${TOKEN}"
```

## 6. 作业提交

### 6.1 查看作业列表

**接口**：`GET /api/assignments?courseId={courseId}`

```bash
curl -X GET "http://localhost:8080/api/assignments?courseId=7" \
  -H "Authorization: Bearer ${TOKEN}"
```

**响应示例**：
```json
{
  "code": 200,
  "data": {
    "records": [
      {
        "id": 33,
        "title": "第3章 作业：实现一个简单的学生管理系统",
        "description": "使用 Java 实现增删改查功能...",
        "dueDate": "2024-01-20T23:59:00",
        "maxScore": 100,
        "submissionStatus": "NOT_SUBMITTED",
        "attachments": [{"name": "作业要求.pdf", "resourceId": 305}]
      }
    ]
  }
}
```

### 6.2 提交作业

**接口**：`POST /api/submissions`

```bash
# 步骤 1：上传作业文件（参照 3.4 上传步骤）
# 获取文件的 fileId

# 步骤 2：提交作业
curl -X POST http://localhost:8080/api/submissions \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "assignmentId": 33,
    "content": "本次作业实现了学生信息的增删改查功能，具体代码见附件...",
    "fileIds": [1024]
  }'
```

**响应**：
```json
{
  "code": 200,
  "message": "作业提交成功",
  "data": {
    "submissionId": 512,
    "assignmentId": 33,
    "submittedAt": "2024-01-18T20:30:00",
    "status": "SUBMITTED"
  }
}
```

> **注意截止时间**：超过截止时间后系统仍可提交，但会标注为"迟交"，是否接受迟交由教师决定。
