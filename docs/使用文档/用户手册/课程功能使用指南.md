## 1. 浏览与加入课程

### 1.1 查看课程列表

**接口**：`GET /api/v1/courses-catalog`

```bash
curl -X GET "http://localhost:8080/api/v1/courses-catalog?page=1&size=10"
```

### 1.2 搜索课程

**接口**：`GET /api/v1/courses-catalog/search?keyword=Java&page=1&size=10`

```bash
curl -X GET "http://localhost:8080/api/v1/courses-catalog/search?keyword=Java&page=1&size=10" \
  -H "Authorization: Bearer ${TOKEN}"
```

### 1.3 查看课程详情

**接口**：`GET /api/v1/courses-catalog/{courseId}`

```bash
curl -X GET http://localhost:8080/api/v1/courses-catalog/7 \
  -H "Authorization: Bearer ${TOKEN}"
```

### 1.4 加入课程

**接口**：`POST /api/enrollments`

```bash
curl -X POST http://localhost:8080/api/enrollments \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{"courseId": 7}'
```

### 1.5 通过邀请码加入课程

**接口**：`POST /api/enrollments/join?code=ABC123`

```bash
curl -X POST "http://localhost:8080/api/enrollments/join?code=ABC123" \
  -H "Authorization: Bearer ${TOKEN}"
```

### 1.6 退出课程

**接口**：`PUT /api/enrollments/courses/{courseId}/drop`

```bash
curl -X PUT http://localhost:8080/api/enrollments/courses/7/drop \
  -H "Authorization: Bearer ${TOKEN}"
```

### 1.7 查看我的选课

**接口**：`GET /api/enrollments?status=ACTIVE`

```bash
curl -X GET "http://localhost:8080/api/enrollments?status=ACTIVE&page=1&size=20" \
  -H "Authorization: Bearer ${TOKEN}"
```

## 2. 查看课程内容

### 2.1 获取课程课节列表

**接口**：`GET /api/v1/lessons/courses/{courseId}`

```bash
curl -X GET http://localhost:8080/api/v1/lessons/courses/7 \
  -H "Authorization: Bearer ${TOKEN}"
```

### 2.2 获取课程资源列表

**接口**：`GET /api/v1/course-resources/courses/{courseId}`

```bash
curl -X GET http://localhost:8080/api/v1/course-resources/courses/7 \
  -H "Authorization: Bearer ${TOKEN}"
```

### 2.3 下载课程资源文件

资源记录会返回 `objectName`，下载时使用文件下载接口：

**接口**：`GET /api/v1/files/download?objectName=...`

```bash
curl -X GET "http://localhost:8080/api/v1/files/download?objectName=uploads/2024/01/abc.pdf" \
  -H "Authorization: Bearer ${TOKEN}" \
  -o "课程讲义.pdf"
```

## 3. 笔记功能

### 3.1 创建笔记

**接口**：`POST /api/notes`

```bash
curl -X POST http://localhost:8080/api/notes \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Java 面向对象笔记",
    "content": "## 核心概念\n\n### 封装\n...",
    "courseId": 7,
    "categoryId": 1
  }'
```

### 3.2 更新笔记

**接口**：`PUT /api/notes/{noteId}`

```bash
curl -X PUT http://localhost:8080/api/notes/88 \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Java 面向对象笔记（更新）",
    "content": "## 核心概念\n\n### 封装\n封装是..."
  }'
```

### 3.3 查看笔记列表

**接口**：`GET /api/notes?courseId={courseId}`

```bash
curl -X GET "http://localhost:8080/api/notes?courseId=7&page=1&size=20" \
  -H "Authorization: Bearer ${TOKEN}"
```

### 3.4 查看笔记历史版本

**接口**：`GET /api/notes/{noteId}/versions`

```bash
curl -X GET http://localhost:8080/api/notes/88/versions \
  -H "Authorization: Bearer ${TOKEN}"
```

### 3.5 恢复笔记版本

**接口**：`POST /api/notes/{noteId}/versions/{versionNo}/restore`

```bash
curl -X POST http://localhost:8080/api/notes/88/versions/2/restore \
  -H "Authorization: Bearer ${TOKEN}"
```

## 4. 课程讨论与问答

### 4.1 查看课程讨论

**接口**：`GET /api/discussions/courses/{courseId}`

```bash
curl -X GET http://localhost:8080/api/discussions/courses/7 \
  -H "Authorization: Bearer ${TOKEN}"
```

### 4.2 发布讨论

**接口**：`POST /api/discussions`

```bash
curl -X POST http://localhost:8080/api/discussions \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "courseId": 7,
    "title": "关于 Java 中 == 和 equals() 的区别",
    "content": "老师您好，我对 == 和 equals() 的区别有疑惑..."
  }'
```

### 4.3 回复讨论

**接口**：`POST /api/discussions`（带 `parentId`）

```bash
curl -X POST http://localhost:8080/api/discussions \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "courseId": 7,
    "parentId": 321,
    "content": "同学你好，== 比较引用，equals() 比较内容..."
  }'
```

## 8. AI 助手

### 8.1 同步对话

**接口**：`POST /api/v1/ai/chat`

```bash
curl -X POST http://localhost:8080/api/v1/ai/chat \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "请解释 Java 中的多态性",
    "courseId": 7
  }'
```

### 8.2 流式对话（SSE）

**接口**：`POST /api/v1/ai/chat/stream`

```bash
curl -N -X POST http://localhost:8080/api/v1/ai/chat/stream \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "请解释 Java 中的多态性",
    "courseId": 7
  }'
```
