# 课伴 功能演示

> 本文档提供 课伴 各功能模块的操作步骤与 API 调用示例，帮助用户快速上手主要功能。

---

## 前置准备

所有受保护的接口均需要在请求头中携带 JWT 令牌：

```bash
Authorization: Bearer <your_token>
```

如需获取令牌，请先完成登录流程。

---

## 一、账号与认证

### 1.1 用户注册

**接口：** `POST /api/v1/auth/register`

```bash
curl -X POST http://localhost:8080/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "zhangsan",
    "password": "123456",
    "studentNumber": "20240042",
    "realName": "张三",
    "role": "STUDENT"
  }'
```

### 1.2 用户登录

**接口：** `POST /api/v1/auth/login`

```bash
curl -X POST http://localhost:8080/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "zhangsan",
    "password": "123456"
  }'
```

### 1.3 获取个人资料

**接口：** `GET /api/users/me`

```bash
curl -X GET http://localhost:8080/api/users/me \
  -H "Authorization: Bearer $TOKEN"
```

### 1.4 更新个人档案

**接口：** `PUT /api/users/me`

```bash
curl -X PUT http://localhost:8080/api/users/me \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "realName": "张三",
    "school": "计算机学院",
    "bio": "热爱学习的大学生"
  }'
```

## 二、课程管理

### 2.1 创建课程（教师）

**接口：** `POST /api/v1/courses-catalog`

```bash
curl -X POST http://localhost:8080/api/v1/courses-catalog \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "code": "CS101",
    "name": "Java 程序设计基础",
    "description": "系统讲授 Java 核心语法与面向对象编程",
    "level": "UNDERGRAD",
    "capacity": 100,
    "thumbnailUrl": "https://example.com/cover.jpg",
    "startDate": "2024-09-01",
    "endDate": "2025-01-15"
  }'
```

### 2.2 搜索课程

**接口：** `GET /api/v1/courses-catalog/search?keyword=Java&page=1&size=10`

```bash
curl -X GET "http://localhost:8080/api/v1/courses-catalog/search?keyword=Java&page=1&size=10" \
  -H "Authorization: Bearer $TOKEN"
```

### 2.3 学生选课

**接口：** `POST /api/enrollments`

```bash
curl -X POST http://localhost:8080/api/enrollments \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "courseId": 1
  }'
```

### 2.4 通过邀请码加入课程

**接口：** `POST /api/enrollments/join?code=ABC123`

```bash
curl -X POST "http://localhost:8080/api/enrollments/join?code=ABC123" \
  -H "Authorization: Bearer $TOKEN"
```

## 三、笔记系统

### 3.1 创建笔记

**接口：** `POST /api/notes`

```bash
curl -X POST http://localhost:8080/api/notes \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Java 变量与数据类型",
    "content": "# Java 数据类型\n\n## 基本数据类型\n- byte、short、int、long\n- float、double\n- char\n- boolean",
    "courseId": 1,
    "categoryId": 1
  }'
```

### 3.2 查看笔记版本历史

**接口：** `GET /api/notes/{noteId}/versions`

```bash
curl -X GET http://localhost:8080/api/notes/1/versions \
  -H "Authorization: Bearer $TOKEN"
```

### 3.3 恢复到指定版本

**接口：** `POST /api/notes/{noteId}/versions/{versionNo}/restore`

```bash
curl -X POST http://localhost:8080/api/notes/1/versions/3/restore \
  -H "Authorization: Bearer $TOKEN"
```

## 四、文件上传

### 4.1 初始化分块上传

**接口：** `POST /api/v1/files/upload/init`

```bash
curl -X POST http://localhost:8080/api/v1/files/upload/init \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "fileName": "large_video.mp4",
    "fileSize": 524288000,
    "contentType": "video/mp4",
    "category": "course-resource"
  }'
```

### 4.2 上传分片

**接口：** `POST /api/v1/files/upload/chunk`

```bash
curl -X POST http://localhost:8080/api/v1/files/upload/chunk \
  -H "Authorization: Bearer $TOKEN" \
  -F "sessionId=${SESSION_ID}" \
  -F "chunkIndex=0" \
  -F "chunk=@/path/to/chunk.part"
```

### 4.3 合并分片

**接口：** `POST /api/v1/files/upload/merge`

```bash
curl -X POST http://localhost:8080/api/v1/files/upload/merge \
  -H "Authorization: Bearer $TOKEN" \
  -F "sessionId=${SESSION_ID}" \
  -F "totalChunks=50"
```

## 五、作业管理

### 5.1 发布作业（教师）

**接口：** `POST /api/v1/assignments/courses/{courseId}`

```bash
curl -X POST http://localhost:8080/api/v1/assignments/courses/1 \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "第一章课后练习",
    "description": "完成教材第一章的所有练习题",
    "dueDate": "2024-12-31T23:59:59",
    "maxScore": 100,
    "attachmentUrl": "https://example.com/assignment.pdf"
  }'
```

### 5.2 提交作业（学生）

**接口：** `POST /api/v1/submissions/assignments/{assignmentId}`

```bash
curl -X POST http://localhost:8080/api/v1/submissions/assignments/1 \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "submissionUrl": "https://example.com/submission.pdf"
  }'
```

### 5.3 批改作业（教师）

**接口：** `POST /api/v1/submissions/{submissionId}/grade`

```bash
curl -X POST http://localhost:8080/api/v1/submissions/1/grade \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "score": 90,
    "comment": "完成质量较高，注意第三题的边界条件处理"
  }'
```

## 六、成绩管理

### 6.1 更新学生成绩（教师）

**接口：** `PUT /api/v1/grades/courses/{courseId}/students/{studentId}`

```bash
curl -X PUT http://localhost:8080/api/v1/grades/courses/1/students/1001 \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "assignmentScore": 88,
    "participationScore": 90,
    "quizScore": 85,
    "midtermScore": 85,
    "finalScore": 90,
    "comments": "期末发挥稳定，继续保持"
  }'
```

### 6.2 查看自己的成绩（学生）

**接口：** `GET /api/v1/grades/courses/{courseId}/my`

```bash
curl -X GET http://localhost:8080/api/v1/grades/courses/1/my \
  -H "Authorization: Bearer $TOKEN"
```
