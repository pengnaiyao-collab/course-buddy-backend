## 🔐 一、认证与用户管理

### 1.1 用户认证

| 功能 | 描述 |
|------|------|
| 用户注册 | 邮箱 + 用户名 + 密码注册，服务端验证唯一性 |
| 用户登录 | 用户名/邮箱 + 密码登录，返回 JWT 令牌 |
| JWT 令牌刷新 | 令牌过期后使用 refresh token 续期 |
| 密码修改 | 登录状态下修改旧密码 |
| 密码重置 | 通过邮件链接重置密码（如已配置邮件服务） |
| 登出 | 服务端令牌失效处理 |

**功能流程：**
```
用户填写注册信息 → 服务端验证（用户名/邮箱唯一性、密码强度）
  → 密码 BCrypt 加密存储 → 返回 JWT 令牌
```

### 1.2 用户档案

| 功能 | 描述 |
|------|------|
| 查看个人档案 | 返回用户名、邮箱、角色、头像等信息 |
| 更新个人档案 | 修改昵称、简介、联系方式等 |
| 头像上传 | 上传图片至 MinIO 并更新头像 URL |
| 角色管理 | ADMIN 可分配/修改用户角色（ADMIN / TEACHER / STUDENT） |

## 📚 二、课程管理

### 2.1 课程基础功能

| 功能 | 描述 |
|------|------|
| 创建课程 | 填写课程名称、描述、封面、分类等基础信息 |
| 编辑课程 | 修改课程元信息（仅课程创建者或 ADMIN） |
| 删除课程 | 软删除课程及关联数据 |
| 发布/下架课程 | 控制课程对学生的可见性 |
| 课程搜索 | 按关键词、分类、状态检索课程 |
| 课程分页列表 | 支持分页、排序展示所有课程 |

### 2.2 课程目录（大纲）

| 功能 | 描述 |
|------|------|
| 创建目录节点 | 添加章/节/课时 |
| 编辑目录顺序 | 拖拽排序章节 |
| 删除目录节点 | 删除指定章节及其课时 |
| 查看课程大纲 | 树形展示课程目录结构 |

## 🗺️ 典型使用场景

### 场景一：学生完整学习路径

```
1. 注册账号 → 完善个人档案
2. 浏览/搜索课程 → 选课加入
3. 查看课程大纲 → 学习课程资源
4. 记录学习笔记（支持分类 & 标签）
5. 参与课程讨论 → 提问 / 回答
6. 使用 AI 助手辅助理解难点
7. 按时提交作业
8. 查看成绩单 & 学习进度统计
9. 与团队成员实时协作编辑笔记
```

### 场景二：教师教学管理路径

```
1. 创建课程 → 编写课程大纲
2. 上传课程资源（PPT / PDF / 视频链接）
3. 发布课程 → 查看选课学生名单
4. 发起课堂签到
5. 发布作业 → 批改作业 → 录入成绩
6. 查看全班学习进度报表
7. 在课程讨论区与学生互动
8. 向学生发送通知消息
```

### 场景三：管理员系统管理路径

```
1. 查看用户列表 → 分配/修改角色
2. 审核教师发布的课程
3. 审核用户上传的内容
4. 查看系统运行状态
5. 导出数据报表
6. 执行数据备份
```
