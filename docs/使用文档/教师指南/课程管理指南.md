## 1. 教师权限概述

教师主要可执行以下操作：

- 创建和维护课程信息
- 管理课节与课程资源
- 发布、更新与批改作业
- 生成签到码、登记出勤
- 维护课程成绩单

---

## 2. 创建与管理课程

### 2.1 创建新课程

**接口**：`POST /api/v1/courses-catalog`

```bash
curl -X POST http://localhost:8080/api/v1/courses-catalog \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "code": "CS101",
    "name": "Java 程序设计基础",
    "description": "系统讲授 Java 核心语法与面向对象编程",
    "level": "UNDERGRAD",
    "capacity": 100,
    "startDate": "2024-09-01",
    "endDate": "2025-01-15"
  }'
```

### 2.2 更新课程信息

**接口**：`PUT /api/v1/courses-catalog/{courseId}`

```bash
curl -X PUT http://localhost:8080/api/v1/courses-catalog/7 \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "description": "更新后的课程描述",
    "capacity": 120
  }'
```

### 2.3 查看我的授课课程

**接口**：`GET /api/v1/courses-catalog/my-teaching`

```bash
curl -X GET "http://localhost:8080/api/v1/courses-catalog/my-teaching?page=1&size=10" \
  -H "Authorization: Bearer ${TOKEN}"
```

### 2.4 删除课程

**接口**：`DELETE /api/v1/courses-catalog/{courseId}`

```bash
curl -X DELETE http://localhost:8080/api/v1/courses-catalog/7 \
  -H "Authorization: Bearer ${TOKEN}"
```

## 3. 课节与资源管理

### 3.1 创建课节

**接口**：`POST /api/v1/lessons/courses/{courseId}`

```bash
curl -X POST http://localhost:8080/api/v1/lessons/courses/7 \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "第1章：Java 入门",
    "description": "介绍 Java 语言历史与环境搭建",
    "lessonOrder": 1,
    "duration": 90
  }'
```

### 3.2 调整课节顺序

**接口**：`POST /api/v1/lessons/courses/{courseId}/reorder`

```bash
curl -X POST http://localhost:8080/api/v1/lessons/courses/7/reorder \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '[101, 102, 103]'
```

### 3.3 添加课程资源

**接口**：`POST /api/v1/course-resources/courses/{courseId}`

```bash
curl -X POST http://localhost:8080/api/v1/course-resources/courses/7 \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "第1章讲义",
    "description": "配套讲义 PDF",
    "resourceType": "PDF",
    "resourceUrl": "https://example.com/resources/ch1.pdf",
    "fileSize": 204800
  }'
```

## 5. 作业管理

### 5.1 发布作业

**接口**：`POST /api/v1/assignments/courses/{courseId}`

```bash
curl -X POST http://localhost:8080/api/v1/assignments/courses/7 \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "第一章课后练习",
    "description": "完成教材第一章的所有练习题",
    "dueDate": "2024-12-31T23:59:59",
    "maxScore": 100
  }'
```

### 5.2 查看课程作业

**接口**：`GET /api/v1/assignments/courses/{courseId}`

```bash
curl -X GET http://localhost:8080/api/v1/assignments/courses/7 \
  -H "Authorization: Bearer ${TOKEN}"
```

### 5.3 发布作业（状态发布）

**接口**：`POST /api/v1/assignments/{assignmentId}/publish`

```bash
curl -X POST http://localhost:8080/api/v1/assignments/33/publish \
  -H "Authorization: Bearer ${TOKEN}"
```

### 5.4 更新作业

**接口**：`PUT /api/v1/assignments/{assignmentId}`

```bash
curl -X PUT http://localhost:8080/api/v1/assignments/33 \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "第一章课后练习（更新）",
    "dueDate": "2025-01-05T23:59:59"
  }'
```

## 6. 作业提交管理

### 6.1 查看作业提交列表

**接口**：`GET /api/v1/submissions/assignments/{assignmentId}`

```bash
curl -X GET http://localhost:8080/api/v1/submissions/assignments/33 \
  -H "Authorization: Bearer ${TOKEN}"
```

### 6.2 下载作业提交压缩包

**接口**：`GET /api/v1/submissions/assignments/{assignmentId}/download`

```bash
curl -X GET http://localhost:8080/api/v1/submissions/assignments/33/download \
  -H "Authorization: Bearer ${TOKEN}" \
  -o "assignment_33_submissions.zip"
```

### 6.3 批改作业

**接口**：`POST /api/v1/submissions/{submissionId}/grade`

```bash
curl -X POST http://localhost:8080/api/v1/submissions/512/grade \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "score": 95,
    "comment": "完成度较高，注意格式规范"
  }'
```

## 7. 出勤管理

### 7.1 生成签到码

**接口**：`POST /api/v1/attendance/courses/{courseId}/code`

```bash
curl -X POST http://localhost:8080/api/v1/attendance/courses/7/code \
  -H "Authorization: Bearer ${TOKEN}"
```

### 7.2 查看当前签到码

**接口**：`GET /api/v1/attendance/courses/{courseId}/code`

```bash
curl -X GET http://localhost:8080/api/v1/attendance/courses/7/code \
  -H "Authorization: Bearer ${TOKEN}"
```

### 7.3 手动登记出勤

**接口**：`POST /api/v1/attendance/courses/{courseId}`

```bash
curl -X POST http://localhost:8080/api/v1/attendance/courses/7 \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "sessionDate": "2024-10-15",
    "entries": [
      {"studentId": 1001, "status": "PRESENT", "remarks": "按时到课"},
      {"studentId": 1002, "status": "ABSENT", "remarks": "请假"}
    ]
  }'
```

### 7.4 查看课程出勤记录

**接口**：`GET /api/v1/attendance/courses/{courseId}`

```bash
curl -X GET http://localhost:8080/api/v1/attendance/courses/7 \
  -H "Authorization: Bearer ${TOKEN}"
```

```bash
    "participationScore": 90,
    "quizScore": 85,
    "midtermScore": 85,
    "finalScore": 90,
    "comments": "期末发挥稳定"
  }'
```
