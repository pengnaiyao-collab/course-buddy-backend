# 认证接口文档

本文档描述 课伴 Backend 的认证相关接口。

---

## 概述

课伴 Backend 使用 **JWT（JSON Web Token）** 进行无状态认证，登录后返回 `accessToken` 与 `refreshToken`。

| 项目 | 说明 |
|------|------|
| 认证方案 | Bearer Token（JWT） |
| 请求头格式 | `Authorization: Bearer <accessToken>` |
| 刷新机制 | `POST /api/v1/auth/refresh-token` |

---

## 接口列表

| 方法 | 路径 | 说明 | 认证 |
|------|------|------|------|
| `POST` | `/api/v1/auth/register` | 注册新用户 | 公开 |
| `POST` | `/api/v1/auth/login` | 用户登录 | 公开 |
| `POST` | `/api/v1/auth/refresh-token` | 刷新 Token | 公开 |
| `POST` | `/api/v1/auth/logout` | 用户登出 | 需要 Token |
| `GET` | `/api/v1/auth/me` | 获取当前用户 | 需要 Token |

---

## POST /v1/auth/register — 注册新用户

### 请求

```json
{
  "username": "zhangsan",
  "password": "Test@123456",
  "studentNumber": "20240042",
  "realName": "张三",
  "role": "STUDENT"
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `username` | string | ✅ | 登录用户名 |
| `password` | string | ✅ | 登录密码 |
| `studentNumber` | string | ❌ | 学号 |
| `realName` | string | ❌ | 真实姓名 |
| `role` | string | ❌ | 角色（默认 STUDENT） |

### 响应（成功）

```json
{
  "code": 200,
  "message": "注册成功",
  "data": {
    "user": {
      "id": 1001,
      "username": "zhangsan"
    },
    "message": "注册成功"
  }
}
```

---

## POST /v1/auth/login — 用户登录

### 请求

```json
{
  "username": "zhangsan",
  "password": "Test@123456"
}
```

### 响应（成功）

```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "user": {
      "id": 1001,
      "username": "zhangsan",
      "role": "STUDENT"
    },
    "accessToken": "eyJhbGciOiJIUzI1NiJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiJ9...",
    "expiresIn": 86400,
    "tokenType": "Bearer"
  }
}
```

---

## POST /v1/auth/refresh-token — 刷新 Token

### 请求

```json
{
  "refreshToken": "eyJhbGciOiJIUzI1NiJ9..."
}
```

### 响应（成功）

```json
{
  "code": 200,
  "message": "Token 刷新成功",
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiJ9...",
    "expiresIn": 86400,
    "tokenType": "Bearer"
  }
}
```

---

## POST /v1/auth/logout — 用户登出

**请求头：** `Authorization: Bearer <accessToken>`

登出会撤销当前用户的 Token。

---

## GET /v1/auth/me — 获取当前用户

**请求头：** `Authorization: Bearer <accessToken>`

```bash
curl -X GET http://localhost:8080/api/v1/auth/me \
  -H "Authorization: Bearer ${TOKEN}"
```

---

## 使用 Token 访问受保护接口

```bash
export TOKEN="eyJhbGciOiJIUzI1NiJ9..."

curl -X GET http://localhost:8080/api/users/me \
  -H "Authorization: Bearer ${TOKEN}"
```
