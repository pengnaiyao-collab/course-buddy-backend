# 核心数据表设计

本章节列出当前核心业务表的关键字段与用途（简版）。如需完整字段，请以 `database/schema.sql` 为准。

---

## 1. 用户表（users）

| 字段 | 说明 |
|------|------|
| `id` | 用户 ID |
| `username` | 登录用户名 |
| `password` | BCrypt 哈希密码 |
| `student_number` | 学号 |
| `real_name` | 真实姓名 |
| `school` | 学院/学校 |
| `role` | 角色（ADMIN/TEACHER/TA/STUDENT） |
| `status` | 状态（ACTIVE/INACTIVE 等） |
| `is_active` | 是否激活 |
| `is_locked` | 是否锁定 |
| `created_at` / `updated_at` | 创建/更新时间 |

---

## 2. 课程表（courses）

| 字段 | 说明 |
|------|------|

## 6. 笔记表（notes / note_versions）

| `notes.content` | 内容 |
