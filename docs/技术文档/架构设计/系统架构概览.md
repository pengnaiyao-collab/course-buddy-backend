# 系统架构概览

## 1. 项目简介

课伴 Backend 基于 Spring Boot 3.2.2 构建，提供课程管理、选课、笔记、作业、考勤、成绩、AI 助手与文件存储等核心能力。系统采用分层架构，结合 JWT 无状态认证与 Redis 缓存，支持 Docker 容器化部署。

## 3. 技术栈详情

### 3.1 核心框架

| 组件 | 版本 | 用途 |
|------|------|------|
| Spring Boot | 3.2.2 | 核心框架，自动配置 |
| Spring Security | 6.x | 认证与授权 |
| Spring Web (MVC) | 6.x | REST API & SSE |
| Spring AOP | 6.x | 横切关注点（审计日志） |
| MyBatis-Plus | 3.5.x | ORM 框架 |
| Flyway | 9.x | 数据库版本迁移 |

### 3.2 数据存储与缓存

| 组件 | 版本 | 用途 |
|------|------|------|
| MySQL | 8.0 | 主关系型数据库 |
| Redis | 7.x | 缓存、Token 黑名单 |
| MinIO | 8.x | 对象存储 |

### 3.3 外部集成

| 组件 | 版本 | 用途 |
|------|------|------|
| XunFei Spark AI | - | AI 对话服务（HTTP + SSE） |

## 6. 请求处理流程

### 6.1 标准 HTTP 请求流程

```
客户端请求
  └─> Nginx 反向代理
      └─> Spring Security Filter
          └─> Controller
              └─> Service
                  └─> Mapper -> MySQL
```

### 6.2 SSE 流式响应

AI 流式对话通过 `SseEmitter` 推送增量内容，客户端使用 `EventSource` 或 `curl -N` 持续接收数据流。
